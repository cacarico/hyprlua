set (modifiers, key, action, arguments, options)
Adds a bind to the internal list.

Parameters:

  • modifiers (string) Modifier keys (e.g., "SUPER")
  • key (string) The key to bind (e.g., "Return")
  • action (string) The action dispatcher (e.g., "exec")
  • arguments (string) Arguments for the dispatcher (e.g., "firefox")
  • options (table) Additional options such as flags and description

Writes all binds to file.
